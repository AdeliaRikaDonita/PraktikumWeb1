sesi 1

Praktek Pemrograman Web 1 akan meembahasa tentang cara menampilkan halaman web dengan menggunakan HTML, CSS dan JavaScript, Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan pemahaman tersebut menuliskan hasil kreasinya sendiri.

  • HTML
  • CSS
  • JavaScript

  1. Pengenalan HTML
  2. Tabel
  3. Form
